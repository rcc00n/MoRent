# More Rent — SEO and content strategy

## 1. SEO goals
Так как трафик идет из Google и Яндекс, SEO должно быть заложено с самого старта, а не потом.

## 2. SEO priorities for MVP
- indexable main pages
- clean URLs
- unique title/description
- semantic headings
- internal linking
- fast loading
- sitemap.xml
- robots.txt
- schema markup on key pages

## 3. Required SEO page groups
### Commercial core
- главная
- каталог авто
- карточки авто
- тарифы
- контакты

### Informational trust/support
- FAQ
- о компании
- условия оплаты и возврата
- политика конфиденциальности

### Phase 2 SEO
- страницы под города / локации
- маршруты
- достопримечательности
- guide pages
- low-frequency landing pages

## 4. Metadata rules
For each page define:
- unique title
- meta description
- H1
- Open Graph title
- Open Graph description
- canonical

## 5. Structured data
Recommended:
- Organization
- LocalBusiness or AutoRental if suitable
- BreadcrumbList
- FAQPage
- Product / Service analogues where appropriate for car pages

## 6. Content principles
- писать по делу
- короткие абзацы
- ясные выгоды
- коммерческий intent на money pages
- trust + details on decision pages
- no keyword stuffing

## 7. Internal linking
Нужно связать:
- главную → каталог
- каталог → карточки авто
- карточки авто → форма заказа
- FAQ → тарифы / условия
- guide pages → relevant car pages
- contacts/about → booking CTA

## 8. Technical SEO checklist
- human-readable slugs
- no duplicate meta
- no empty H1
- responsive pages
- optimized images
- correct status codes
- 404 page
- sitemap generation
- robots config

## 9. Content backlog for phase 2
- аренда авто в [город/регион]
- авто для маршрутов по региону
- советы по выбору авто под поездку
- FAQ по аренде в регионе
- маршруты и достопримечательности
